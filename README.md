# 📉 Telecom Churn Prediction (End-to-End ML Project)

## 📌 Business Problem

Customer churn is a major challenge in the telecom industry. Retaining existing customers is more cost-effective than acquiring new ones. This project aims to predict customers who are likely to churn.

* Contains customer demographics, services, and account information
* Target variable: Churn (Yes/No)
