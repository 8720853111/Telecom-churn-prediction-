# 📉 Telecom Churn Prediction (End-to-End ML Project)

## 📌 Business Problem

Customer churn is a major challenge in the telecom industry. Retaining existing customers is more cost-effective than acquiring new ones. This project aims to predict customers who are likely to churn.

---

## 📊 Dataset Overview

* Contains customer demographics, services, and account information
* Target variable: Churn (Yes/No)

---

## 🔍 Key Steps

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Building

---

## 🤖 Models Used

* Decision Tree
* Random Forest
* RFE

---

## 📈 Model Performance

| Model             | Accuracy | Remarks              |
| ----------------- | -------- | -------------------- |
| Decision Tree     | 86%      | Best performance     |
| Random Forest     | 80%      | Improved performance |
| RFE              | 78%      | Baseline model       |

---

## 📊 Key Insights

* Customers with high monthly charges are more likely to churn
* Contract type significantly impacts churn rate
* Long-term customers are less likely to churn
